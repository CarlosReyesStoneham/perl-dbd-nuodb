Welcome to the perl-dbd-nuodb Project

This project provides a Perl DBI compatible DBD module for the NuoDB database.

Sample

The following illustrates how to use the NuoDB DBD module in an application:


      use DBI;
      my $dbh = DBI->connect("dbi:NuoDB:".$database.'@'.$host':'.$port, $username, $password, {schema => $schema});
      my $sth = $dbh->prepare("SELECT 'one' FROM DUAL");
      $sth->execute();
      my ($value) = $sth->fetchrow_array();
